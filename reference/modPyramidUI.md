# Age-Sex Pyramid Module - UI Function

Age-Sex Pyramid Module - UI Function

## Usage

``` r
modPyramidUI(id)
```

## Arguments

- id:

  character vector of length 1. Module namespace identifier.

## Value

A `div` containing age-sex pyramid UI.
