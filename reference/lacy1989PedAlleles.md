# lacy1989PedAlleles is a dataframe produced by `geneDrop` on `lacy1989Ped` with 5000 iterations.

lacy1989PedAlleles is a dataframe produced by `geneDrop` on
`lacy1989Ped` with 5000 iterations.

## Usage

``` r
lacy1989PedAlleles
```

## Format

An object of class `data.frame` with 14 rows and 5002 columns.

## Source

lacy1989Ped is a dataframe containing the small example pedigree used by
Robert C. Lacy in "Analysis of Founder Representation in Pedigrees:
Founder Equivalents and Founder Genome Equivalents" Zoo Biology
8:111-123 (1989).
