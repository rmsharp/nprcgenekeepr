# qcBreeders is a list of 29 baboon IDs that are potential breeders

qcBreeders is a list of 29 baboon IDs that are potential breeders

## Usage

``` r
qcBreeders
```

## Format

An object of class `character` of length 29.

## Source

qcBreeders is a list of 3 males and 26 females from the `qcPed` data
set.
