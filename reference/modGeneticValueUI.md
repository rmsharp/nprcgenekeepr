# Genetic Value Analysis Module - UI Function

Genetic Value Analysis Module - UI Function

## Usage

``` r
modGeneticValueUI(id)
```

## Arguments

- id:

  character vector of length 1. Module namespace identifier.

## Value

A `div` containing genetic value analysis UI.

## References

Vinson, A. and Raboin, M.J. (2015) *Journal of the American Association
for Laboratory Animal Science*, **54**(6), 700-707.
