# getVersion Get the version number of nprcgenekeepr

getVersion Get the version number of nprcgenekeepr

## Usage

``` r
getVersion(date = TRUE)
```

## Arguments

- date:

  A logical value when TRUE (default) a date in YYYYMMDD format within
  parentheses is appended.

## Value

Current Version

## Examples

``` r
library(nprcgenekeepr)
getVersion()
#> [1] "1.1.0.9000 (2026-06-17)"
```
