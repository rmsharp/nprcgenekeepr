# Get the version number of nprcgenekeepr

Get the version number of nprcgenekeepr

## Usage

``` r
getVersion(date = TRUE)
```

## Arguments

- date:

  A logical value when TRUE (default) a date in YYYY-MM-DD (ISO) format
  within parentheses is appended.

## Value

Current Version

## Examples

``` r
library(nprcgenekeepr)
getVersion()
#> [1] "2.0.0.9000 (2026-07-21)"
```
