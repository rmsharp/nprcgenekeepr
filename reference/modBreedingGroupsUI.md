# Breeding Groups Module - UI Function

Breeding Groups Module - UI Function

## Usage

``` r
modBreedingGroupsUI(id)
```

## Arguments

- id:

  character vector of length 1. Module namespace identifier.

## Value

A `div` containing breeding group formation UI.

## References

Vinson, A. and Raboin, M.J. (2015)
