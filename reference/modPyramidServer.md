# Age-Sex Pyramid Module - Server Function

Age-Sex Pyramid Module - Server Function

## Usage

``` r
modPyramidServer(id, pedigreeData)
```

## Arguments

- id:

  character vector of length 1. Module namespace identifier.

- pedigreeData:

  reactive returning pedigree data frame.

## Value

List with `data`, `plot`, and `livingCount`.
